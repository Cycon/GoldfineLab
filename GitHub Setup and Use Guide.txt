****Setting Up GitHub/Setting up local repository (Applies to Everybody)
1. Open GitHub Client and Log In
2. Go to the repository page in your internet browser, https://github.com/Cycon/GoldfineLab
3. Press "Clone in Desktop" (bottom right hand corner). This will clone the master branch to a new folder on your desktop.

 
****Accessing Latest Code (Non-coders/Reading or compiling code)
1. Open GitHub Client and Sign In.
2. Press "Sync" in the upper right hand corner.
3. Code is now updated to the newest master version in your local directory. 
4. Run and compile code as you would normally.


****Setup(New Coders/New Features)
When starting Github, you will likely have your active branch be the master. 
1. To start editing code, change this branch to your personal one or create a new one.
	1a. To create a new branch, go to the top of GUI where it says "master" with a dropdown arrow.
	1b. Click this, it should open the list of branches and a search bar.
	1c. Type in the name of your new branch (either a feature name or your personal name)
	    -Make it a feature name if you plan on coding with someone else
	    -Otherwise, make a personal branch to use for your personal code revisions/testing.
